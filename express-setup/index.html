Express 환경 설정

package.json 생성

npm init -y

scripts 실행

"scripts": {
    "dev": "node index.js"
}

npm run dev

express 설치

npm install express

package.json

  "dependencies": {
    "express": "^4.17.3"
  }

Babel 설치

babeljs.io : 최신 자바스크립트를 NodeJS가 이해하는 자바스크립트로 안전하게 컴파일해서 변환

npm install --save-dev @babel/core @babel/preset-env @babel/node

babel.config.json

{
  "presets": ["@babel/preset-env"]
}

실행할 때

"scripts": {
    "dev" : "babel-node index.js"
}

Nodemon

nodemon을 설치하면 수정되는 걸 감시

npm install nodemon --save-dev

"scripts": {
    "dev": "nodemon --exec npx babel-node index.js"
}

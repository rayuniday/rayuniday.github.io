Vue CLI로 Vue 프로젝트 생성하기

Vue CLI 설치

Vue CLI는 Vue 프로젝트를 빠르게 구성하고, 빌드, 디플로이 할 수 있게 도와주는 도구다. 여기서 CLI는 Command Line Interface의 약자로 터미널에 명령어를 입력하여 컴퓨터와 상호 작용하는 방식을 의미한다.

그럼 Vue CLI를 설치해보겠다. Vs code의 터미널에 다음 명령어를 입력하여 Vue CLI를 설치한다.

npm install -g @vue/cli
← 하위 컴포넌트에서 상위 컴포넌트로 데이터 전달하기
